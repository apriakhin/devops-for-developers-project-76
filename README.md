## Getting Started

### Сheck the availability of servers

```bash
make ping
```

### Install the requirements

```bash
make install
```

### Setup the environment

```bash
make setup
```

### Run the deployment

```bash
make deploy
```

### Setup the monitoring

```bash
make monitoring
```
